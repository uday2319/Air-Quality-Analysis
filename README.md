 **Project Overview:**
Developed an Air Quality Analysis and Visualization project using Python libraries like NumPy, Pandas, and Matplotlib to study AQI trends for the year 2025 across different global cities.

 **Key Highlights:**

-> Collected and analyzed air quality data focusing on pollutants such as PM2.5 and PM10, which impact human health.

-> Used Pandas for data cleaning, manipulation, and handling large datasets.

-> Applied NumPy for numerical analysis, including averages and standard deviations.

-> Created visualizations with Matplotlib to show trends, seasonal patterns, and pollution levels across regions.

-> Compared results against WHO standards to identify cities with critical pollution levels.

-> Analyzed time-series data to detect pollution spikes caused by traffic, industries, or wildfires.

 **Outcome: **
The project provided valuable insights into global air pollution trends and highlighted areas where policy intervention is needed.
Gained hands-on experience in data cleaning, visualization, and environmental data analysis.
